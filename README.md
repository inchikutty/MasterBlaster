# MasterBlaster
Comp 680 Game using phaser
=======
This is a project for COMP 680 class at CSUN Spring 2016.
Objective is to build a game "master-blaster 10000" with phaser game engine.


Tools used
----------

yeoman, gulp, phaser, npm, browserify
